# 🔐 Secrets Project

A web application built with **Node.js**, **Express.js**, **Axios**, and **EJS** that fetches and displays random secrets from an external API. The application demonstrates server-side rendering, API integration, and dynamic content generation.

## 🚀 Features

* Fetches random secrets from an external API
* Displays secret messages along with usernames
* Server-side rendering using EJS templates
* Static asset management with Express
* Responsive and clean user interface
* Error handling for API requests

## 🛠️ Technologies Used

* Node.js
* Express.js
* Axios
* EJS
* HTML
* CSS

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/anushachinthakayala0064/secrets-project.git
```

2. Navigate to the project directory:

```bash
cd secrets-project
```

3. Install dependencies:

```bash
npm install
```

4. Start the application:

```bash
nodemon index.js
```

5. Open your browser and visit:

```
http://localhost:3000
```
